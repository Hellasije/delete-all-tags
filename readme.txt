# Delete All Tags WordPress Plugin

## Description
Delete All Tags is a WordPress plugin that allows you to delete all tags from your WordPress site with a single click. 

**Warning:** This action is permanent and cannot be undone.

## Installation
1. Download the plugin zip file.
2. Upload the plugin to your WordPress site's plugin directory (`wp-content/plugins/`).
3. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage
1. Once activated, you will see a new menu item named 'Delete All Tags' in your WordPress admin menu.
2. Click on 'Delete All Tags' to navigate to the plugin page.
3. You will see the number of tags that will be deleted.
4. Click on the 'Delete Tags' button to proceed. 
   **Note:** This action is irreversible.
